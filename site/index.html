Layout

A simple grid. Flexible and responsive.

Build dynamic columns using the .col class, or definitive widths by suffixing col with your width, like .col-30. Wrap your columns in a .row class, with optional gutters.
There are even classes like .responsive-sm, to break the grid at specific breakpoints.
Resize your browser window to see it in action.

Positioning

Based on a simple $space variable, and a set of helper classes, positioning your elements with padding or extra margin is as easy as adding a class. In this case, $space is equal to 1em.

Side Class Description
All .pad / .mar Space all the things.
Horizontal .pad--h / .mar--h Space the left and right.
Vertical .pad--v / .mar--v Space the top and bottom.
Top .pad--t / .mar--t Space the top.
Bottom .pad--b / .mar--b Space the bottom.
Left .pad--l / .mar--l Space the left.
Left .pad--l / .mar--l Space the right.

Dropdown menu

Vanilla JavaScript, totally naked dropdown component. Add the data-idea-toggle attribute to a container like a button. Inside said button, add the attribute data-idea-target to the element you would like to toggle. On click, a class -is-visible will be added to the data-idea-target. 

<button class="p-menu--toggle" data-idea-toggle>
    Dropdown ▾
    <menu class="p-menu" data-idea-target>
        <li>Menu Item</li>
        <li>Another One</li>
        <li>More Items</li>
        <li>Last Item</li>
    </menu>
</button>

Color System

Generate a starting point for your color system by adding your primary/project/brand color to the _colors.scss file as the $color-primary variable.

Using a little Sass magic, your color palette will be generated to fit your shit.
