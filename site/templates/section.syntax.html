rapid prototyping syntax

The syntax has been formulated for optimal typing speed. A class system based on common patterns for layout, spacing, positioning, typography and even common design patterns, like cards. Classnames are aptly named using an easy-to-memorize syntax, and most class functions are atomic in structure.

All positioning and spacing classes use a variable called $space which represents 1rem. The font-size is set to 100% on the body tag, meaning $space = 1rem = 16px.
