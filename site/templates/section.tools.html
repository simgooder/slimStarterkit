toolkit

This grunt task-set has everything you need to get started (static-file server, sass-compiler) set up based around the existing folder structure, so all you have to do is type in a few grunt commands, and you're ready to roll.

Features
  • Static file server - grunt-web-server
  • Sass Compiler - grunt-sass
  • Autoprefixer - Post-CSS
  • Browser refreshing / Style-injection - Browsersync
  • HTML templating system - grunt-bake
  • File Watch - grunt-contrib-watch

Directions
  1. Open Terminal
  2. Navigate to your project folder root
  3. Type `grunt server`. This will start your static file server
  4. Open another terminal window in the folder root and type `grunt`
  5. Start coding

With these two processes running, you can now code without worrying about refreshing the page, compiling your Sass, or prefixing your code. Follow the same process every time to spin up a server and start coding in no time.
